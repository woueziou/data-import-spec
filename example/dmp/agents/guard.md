# Sentinel

Track uncertainty, drift, validation, and quality.

This agent should cover the documented work from steps 8, 9, and 11.

Create:
- confidence notes
- ambiguity log
- drift rules
- drift detection plan
- schema versioning plan
- quality checks
- quality rules definition
- quarantine and retry guidance
- validation function spec
- parser blockers
- low-confidence classification follow-ups
- consumer requirements table
- consumer validation gaps
- do-and-don't list
- edge-case and risk matrix
- maintainability notes
- security and robustness guardrails

Loop back when the source changes or a rule fails.

Handoff: `dmp-serve`
